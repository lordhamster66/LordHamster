## 安装依赖包

```bash
npm install stylelint stylelint-config-standard stylelint-order stylelint-scss postcss-scss -D
```

## 在package.json中新增命令
```json
{
	...
	"lintcss": "stylelint src/**/*.{css,scss} --fix --custom-syntax postcss-scss"
}
```
然后执行`npm run lintcss`，就能够校验`css`代码了。

## 与vscode联动

### 安装vscode插件stylelint
![](https://img.lordhamster.com/img/2022/10/01/20221001143749.png)### 修改vscode的setting.json文件：
```json
"stylelint.validate": ["css", "scss"],
"stylelint.customSyntax": "postcss-scss",
"editor.codeActionsOnSave": {
	"source.fixAll.stylelint": true
 }
```
至此，在vscode中保存`css`代码就能自动校验了。
