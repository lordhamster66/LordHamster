# Config

## 1.创建密码;

```bash
In [1]: from notebook.auth import passwd
In [2]: passwd()
Enter password: ******
Verify password: ******
Out [2]: 'sha1:...'
```

保存 out 后面的密钥

## 2.生成配置文件:

```bash
jupyter notebook --generate-config
```

## 3.修改配置文件:

```bash
vim ~/.jupyter/jupyter_notebook_config.py
```

在文件末尾加入如下内容:

```bash
c.NotebookApp.allow_remote_access = True #允许远程连接
c.NotebookApp.ip='*' # 设置所有ip皆可访问
c.NotebookApp.password = u'sha:..' #之前复制的密码
c.NotebookApp.open_browser = False # 禁止自动打开浏览器
c.NotebookApp.port =8888 #任意指定一个端口
c.NotebookApp.default_url = '路径' #添加存储路径
```

# Run

## 直接启动:

```bash
jupyter notebook
```

## 后台运行:

```bash
nohup jupyter notebook >~/jupyter.log 2>&1 &
```

停止

```bash
ps -aux |grep jupyter  # 找到jupyter notebook进程杀死即可
kill -9 pid
```

# Customize

## [jupyter-themes](https://github.com/dunovank/jupyter-themes)

### Install with pip

```bash
# install jupyterthemes
pip install jupyterthemes

# upgrade to latest version
pip install --upgrade jupyterthemes
```

### Install with conda

```bash
# install jupyterthemes
conda install -c conda-forge jupyterthemes

# update to latest version
conda update jupyterthemes
```

### Use theme

```bash
jt -t monokai -fs 95 -altp -tfs 11 -nfs 115 -cellw 88% -N -T
```
