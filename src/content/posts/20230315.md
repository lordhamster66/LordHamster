## 查看显卡型号
```bash
ubuntu-drivers devices
```
![](https://img.lordhamster.com/img/2023/03/16/20230316195713.png)
## 安装指定版本
```bash
sudo apt install nvidia-driver-515-server
```
## 安装完成重启
```bash
nvidia-smi
```
